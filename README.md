<p>how to create saza app :</p>

```sh

npm i -g saza
saza my-app-name

```

for typescript :
```sh

npm i -g saza
saza my-app-name --typescript 

```

or <code>npx</code>

```sh

npx my-app-name --typescript 

```


for dev live server :
```sh

npm run start

```

build production from saza app :
```sh

npm run build

```
